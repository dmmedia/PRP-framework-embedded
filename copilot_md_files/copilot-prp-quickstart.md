# Copilot PRP Quickstart

This guide explains how to use the PRP framework with GitHub Copilot and VS Code.

## Getting Started
1. Install the recommended extensions (see .vscode/extensions.json)
2. Open a PRP plan file and follow the instructions
3. Use Copilot Chat to execute PRP workflows

## Key Differences from Claude
- No external CLI required
- Prompts are Copilot-compatible
- All validation and reporting in-editor

See also: [Copilot Migration Guide](copilot-migration-guide.md)
